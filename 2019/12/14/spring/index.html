Spring学习笔记

框架的概念：

  1. 高度抽取可重用代码的设计
  2. 高度的通用性

Spring的特点:

优良特性：

  1. 非侵入式
  2. 依赖注入
  3. 面向切面编程
  4. 容器
  5. 组件化
  6. 一站式

Core Container（IOC）核心容器包含Bean Core Context expression 这几个容器

Web：Sping开发Web应用

  1. WebSocket
  2. Servlet
  3. web protlet

    (建议用哪个模块就导哪个包)

IOC：

控制反转;被动式的资源获取方式：资源的获取不是自己创建，而是交给容器创建和设置

DI：（Injevtion 依赖注入）

  1. 容器能知道哪个组件（类运行的时候时候需要另外一个类）
  2. 只要使用管理的组件，都能使用组件提供的强大功能。现在所有的对象交给容器创建，容器中的组件注册。
